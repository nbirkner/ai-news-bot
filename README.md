# ai-news-bot

A GitHub Actions bot that monitors top AI/tech newsletters and posts to Slack.

- **Breaking news:** Checks every 2 hours. Posts immediately when a tier-1 source (Anthropic, OpenAI, NVIDIA, etc.) publishes something matching a high-signal keyword.
- **Daily digest:** Posts at 8am PST with all non-breaking articles from the last 24h.

Summaries are generated by Together AI's inference API (`meta-llama/Meta-Llama-3-8B-Instruct-Lite`).

## Setup

### 1. Fork or clone this repo into your GitHub organization

### 2. Create a Slack Workflow Builder webhook

In Slack: Automations → Workflow Builder → New Workflow → Starts with a webhook.
Copy the webhook URL.

### 3. Add GitHub Secrets

Go to Settings → Secrets and variables → Actions, and add:

| Secret | Value |
|--------|-------|
| `TOGETHER_API_KEY` | Your Together AI API key (together.ai/settings) |
| `SLACK_WEBHOOK_URL` | The Workflow Builder webhook URL from step 2 |
| `GH_PAT` | A Personal Access Token with `repo` scope (for committing state) |

### 4. Enable workflows

Workflows are paused by default on forks. Go to the Actions tab and enable them.

### 5. Test manually

Run either workflow from the Actions tab using "Run workflow" to verify your setup before the first scheduled run.

## Configuration

Edit `config.yaml` to:
- Add or remove news sources
- Adjust breaking news keywords
- Change the Together AI model
- Modify how many digest items are included (`max_digest_items`)

No code changes needed — all tunable knobs are in config.

## Adding a new source

In `config.yaml`, add an entry under `sources.standard` (or `sources.tier1` for breaking-eligible):

```yaml
- name: "My Newsletter"
  url: "https://mynewsletter.com/feed"
  type: rss
  domain: "mynewsletter.com"
```

## Local development

```bash
python -m venv .venv && source .venv/bin/activate
pip install -r requirements.txt
export TOGETHER_API_KEY=...
export SLACK_WEBHOOK_URL=...
python dry_run.py          # safe test — no Slack posting
python main_breaking.py    # requires TOGETHER_API_KEY + SLACK_WEBHOOK_URL
python main_digest.py
```

## Running tests

```bash
pytest tests/ -v
```
